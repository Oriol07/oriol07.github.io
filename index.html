Warcraft Game Companion


Android Studio
Kotlin
Firebase
University


This university project was a game companion for World of Warcraft (unofficial, for learning). Players could upload and share their warcraft photos, similar to Instagram. I used a model-viewmodel-view architecture. I added features like handling errors, Facebook login, using multiple tasks at once with coroutines, and making things respond quickly using LiveData. I also connected the app with the internet by using Firestore as a database.




Llengua màgica: the RPG for learning Catalan


Unreal Engine
C++
Blueprints
University


The game, teaching Catalan through English, was an experiment exploring how RPGs can teach languages, inspired by other educational games. Throughout development, I focused on creating AI for enemies using behavior trees, which was enjoyable and insightful.
